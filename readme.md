# NewStatSVN
***
[![](https://img.shields.io/aur/license/:package.svg)]()
[![](https://img.shields.io/badge/java-1.8-blue.svg)]()
***
* 使用svn命令行导出指定仓库的xml日志
* 使用java命令运行可执行的jar文件
* 上传xml日志到可执行jar服务中
* 可视化的浏览该项目的各个数据指标
    目前可用的视图：
        雷达图-指定开发者的提交以及修改的文件类型
        饼图-该项目的开发者及其提交所占整体项目的比例
        年度提交热度图-指定开发者在一年中对项目的贡献次数（目前采用提交代码的次数计算）
